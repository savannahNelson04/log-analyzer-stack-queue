Log Analyzer Using Sacks and Queues <br>

Description <br>
This project implements a log processing system that analyzes 10,000+ log entries using stacks and queues. It extracts insights such as error tracking, log distribution, and memory warnings.
<br>

Features
- Queue-based log processing
- Stack-based error tracking
- Log level analysis
- Memory warning detection
